# Task2 第一个Python程序

### 学习目标

1. 了解什么是命令行模式和Python交互模式。
2. 初步掌握Python的输入及输出



### 学习任务

1. 编写一个程序，要求：输入矩形形的长和宽，输出矩形面积

   输入示例

   ```
   3 6
   ```

   输出示例

   ```
   The area is 18, haoye!
   ```
